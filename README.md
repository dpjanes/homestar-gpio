# homestar-gpio
IOTDB / HomeStar Controller for GPIO (e.g. Raspberry Pi, BeagleBone, Intel Edison…)

# Installation

Install Home☆Star first. 
See: https://github.com/dpjanes/iotdb-homestar#installation

Then

    $ homestar install homestar-gpio
